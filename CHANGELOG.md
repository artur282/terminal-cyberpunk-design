# Changelog

All notable changes to the **Terminal Cyberpunk Design** skill will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-03-05

### Added

- Initial release of the Terminal Cyberpunk Design skill
- **SKILL.md** — Core design system with 6 visual pillars
- **Design Tokens** — Complete CSS custom property system (backgrounds, borders, colors, typography, spacing, animations)
- **Components Library** — Cards, tables, log panels, navigation, stats, CTAs, inputs
- **Typography Guide** — Curated font pairings with 5 role categories (display, body, hero, retro, technical)
- **Landing Page Example** — Full working reference implementation (1000+ lines)
- 5 terminal archetypes (Military ops, Hacker underground, Corporate sci-fi, Retro phosphor, Neural interface)
- 6 accent color palettes (Phosphor Green, Cyan Protocol, Amber Terminal, Crimson Alert, Violet Synthwave, Arctic Mint)
- Atmosphere effects (grid HUD, noise texture, scanlines, radial glow, vignette)
- Decorative elements (corner brackets, data annotations, typing indicators, ASCII art)
- Animation system (staggered reveals, glitch effects, typing simulation, scroll-triggered)
- Visual naming conventions table
- Anti-patterns documentation

### Design Philosophy

- Established "not a generic dark theme" principle
- Defined memorable "one thing" requirement per design
- Created syntax-highlighting inspired color semantics
